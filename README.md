# אתר עמותת אור הנפש

אתר סטטי בעברית, RTL, לעמותה תורנית העוסקת בתורה, הוראה ותורת הנפש.

## עמודים

- `index.html` - דף הבית
- `about.html` - אודות
- `donate.html` - תרומה חד פעמית והוראת קבע חודשית
- `books.html` - רכישת ספרי הרב
- `spark.html` - ניצוץ של קדושה
- `qna.html` - שאלות ותשובות
- `publish.html` - פרסום בדף הבית של בית הוראה ובית דין
- `ask-rabbi.html` - שאל את הרב
- `women.html` - הפינה הנשית
- `soul-torah.html` - תורת הנפש

## חיבורים עתידיים

- סליקה: לחבר את טפסי התרומה והספרים לספק סליקה ישראלי מאובטח.
- מייל: לחבר את טופס "שאל את הרב" לכתובת המייל האמיתית של הרב, עם הגנת ספאם.
- תוכן: להחליף את טקסטי הדוגמה בתוכן, ספרים, מחירים ופרטי קשר אמיתיים.
- ניהול תוכן: אם רוצים לעדכן מאמרים/שאלות/פרסומים בקלות, מומלץ לחבר CMS.

## קבצים מרכזיים

- `styles.css` - העיצוב הרספונסיבי המלא
- `script.js` - תפריט מובייל, אנימציות, FAQ, תרומה וסל ספרים
- `assets/` - איורי רקע קלים ב-SVG
